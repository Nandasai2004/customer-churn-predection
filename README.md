📊 Customer Churn Prediction


🚀 Overview

Customer Churn Prediction is a machine learning project that predicts whether a customer is likely to leave a service. This helps businesses take proactive steps to improve customer retention and reduce revenue loss.


🎯 Objectives
Predict customer churn using ML models
Identify key factors affecting churn
Enable data-driven business decisions

📂 Dataset

The dataset includes:

Customer demographics

Account details (tenure, services)

Billing information (monthly & total charges)

Churn label (Yes/No)


⚙️ Tech Stack

Language:  Python 🐍
Libraries: NumPy , Pandas , Matplotlib , Seaborn , Scikit-learn
